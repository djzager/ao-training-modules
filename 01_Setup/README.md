# Setup

## Prerequisites

1. `ansible` 2.6+
1. `openshift` Python package 0.6.0+
1. `python` 2.7+
1. `operator-sdk` 0.0.7+

Optional:

1. `ansible-runner` 1.0+
1. `ansible-runner-http` 1.0+

## Installation
```
$ sudo dnf install ansible python2-openshift python2 
```

Optional:
```
$ pip install ansible-runner ansible-runner-http
```
